# NEXTFLOW-WGS-DEMO-PIPELINE

## Overview

This repository contains a demonstration bioinformatics pipeline for analyzing Whole Genome Sequencing (WGS) data using Nextflow.

The workflow simulates a standard genomic analysis pipeline including read quality control, alignment, variant calling, and result generation.

The goal of this project is to demonstrate reproducible and scalable bioinformatics workflows using Nextflow.

---

## Pipeline Workflow

1. Quality Control of sequencing reads
2. Alignment of reads to reference genome
3. Variant calling
4. Generation of analysis outputs

---

## Tools Used

* Nextflow
* FastQC
* BWA
* SAMtools
* BCFtools
* Bash
* Linux command line

---
